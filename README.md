Ecom Product Catalog

Overview

This is a full-stack product catalog application built using Spring Boot for the backend and React (Vite) for the frontend. The system provides basic product listing and category filtering functionality through REST APIs.

The project is structured as two independent modules:

A backend service responsible for APIs and data handling
A frontend application responsible for UI and user interaction
Tech Stack

Backend:

* Java
* Spring Boot
* Spring Data JPA
* Maven

Frontend:

* React (Vite)
* JavaScript
* HTML
* CSS
